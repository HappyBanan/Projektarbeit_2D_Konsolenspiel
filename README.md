# Labyrinthspiel in C

## Voraussetzungen
- GCC (C11)
- Terminal (UTF-8)

## Build
```sh
make                                // Konsoleneingabe für Gameerstellung / Neukalibrierung / Updateübernahme
```

## Ausführen
```sh
./labyrinth                         // Konsoleneingabe für Gamestart
```

## Steuerung && Anzeige (Konfigurierbar [main.c (6-22)])
- **W / w** : hoch
- **A / a** : links
- **S / s** : runter
- **D / d** : rechts
- **1**     : Restart
- **0**     : Beenden

- **P**     : Spieler
- **T**     : Schatz
- **O**     : Hindernis
- **.**     : Freifläche
